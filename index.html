Get started

Let the magic happen

Create a new project

npx bananasplit-js new "my-app" --git|-g

or

Download

Build the stack

Add your environment variables to .env

yarn build:stack | npm run build:stack
Ready to dev

How it works

Easy and fast as you can't imagine


# [database]

DB_DIALECT=mysql

DB_HOST=localhost
DB_PORT=3306

DB_DATABASE=my-backend
DB_USERNAME=bananasplit
DB_PASSWORD=secret

Set your environment variables

Add your database engine, your database name (no need to create it), and the credentials.

 

# choose your package manager

yarn build:stack
npm run build:stack

Run the stack builder

It will detect your package manager and database to install Express, Typescript, Sequelize, Jest, your Database Driver and more! Also it will run an integration test to check the setup

 

# which is yours?
yarn dev
npm dev

Start the development server

The services are already written. Your database is already created. Just run the development server and code the business!

 

yarn build
yarn route:list
yarn generator:create
yarn test:watch
yarn upgrade:stack
yarn lint:fix

¿Need to perform an action?

The swiss knife of scripts. Just pick up what you need: Run the route and middleware inspector, create resources in command-line to use in your REST client, run tests in coverage mode, and more...
